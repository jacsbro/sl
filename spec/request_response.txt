Suche:
=============
Request: /search/
Parameter:
- Anfragestring mit AND OR ?
- aktive kategorien[]
  pro kategorie
    - id
    - name
    - wert
- startwert (paging)
- seitengröße
- ranking (DWD bzw. default)

Response:
- Kategorie[] (sortiert nach rank des Thesaurus)
  pro Kategorie:
    - id
    - name
    - keyword[]
      pro keyword:
      - id
      - name
      - count

- Anzahl Treffer
- Treffer[]
  pro Treffer:
  - id
  - titel
  - beschreibung
  - produkttyp
  - kategorie[]
    pro kategorie:
      - id
      - name
      - keywords[]
  - ol-resourcen[]
